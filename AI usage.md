# AI Usage Policy

Copyright 2026 Jiří Šach

## Statement

This project uses AI-assisted development tools including Claude (Anthropic)
and GitHub Copilot (GitHub/Microsoft) as coding assistants.

## Scope of AI Assistance

AI tools were used to assist with:
- Code generation and refactoring suggestions
- Documentation drafting

AI tools were NOT used for:
- Architectural decisions
- Security model design
- Business logic and policy rules
- Test case design and invariant definitions

## Author Responsibility

All code in this repository has been:
- Reviewed by the author
- Modified and adapted as needed
- Tested against the project's test suite
- Integrated with deliberate architectural decisions

Architectural decisions are documented by the author
and represent original design choices.
